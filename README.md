## QUALITY CHECKLIST

Use this checklist after generating content:

### SEO Checklist
- [ ] Primary keyword in meta title (near beginning)
- [ ] Primary keyword in meta description
- [ ] Primary keyword in H1 (exactly once)
- [ ] Primary keyword in first 100 words
- [ ] Primary keyword in 2-3 H2 headings naturally
- [ ] Primary keyword in conclusion
- [ ] Secondary keywords used throughout
- [ ] Keyword density between 0.5-2%
- [ ] No keyword stuffing
- [ ] All images have descriptive alt text
- [ ] Internal links included (3-5 relevant pages)
- [ ] External links to authority sources (2-3)
- [ ] URL is short and keyword-optimized
- [ ] Meta title under 60 characters
- [ ] Meta description under 160 characters

### Content Quality Checklist
- [ ] Answers the search intent completely
- [ ] More comprehensive than top competitors
- [ ] Unique angle or insight included
- [ ] Actionable advice/steps provided
- [ ] Examples and data included
- [ ] Easy to read (short paragraphs, subheadings)
- [ ] No fluff or filler content
- [ ] Accurate and up-to-date information
- [ ] Proper grammar and spelling
- [ ] Consistent brand voice

### Structure Checklist
- [ ] Clear heading hierarchy (H1 → H2 → H3)
- [ ] Subheadings every 200-300 words
- [ ] Short paragraphs (2-4 sentences max)
- [ ] Bullet points where appropriate
- [ ] Numbered lists for steps/processes
- [ ] Tables for comparisons
- [ ] Visual breaks (images, callouts)
- [ ] Logical flow from intro to conclusion

### Conversion Checklist
- [ ] Clear value proposition
- [ ] Benefits emphasized over features
- [ ] Social proof included (stats, testimonials)
- [ ] Objections addressed
- [ ] Clear primary CTA
- [ ] Secondary CTA option
- [ ] CTA above fold and at end
- [ ] Urgency/scarcity if appropriate

### Technical Checklist
- [ ] FAQ section with schema-ready format
- [ ] Featured snippet opportunities optimized
- [ ] Table of contents for long content
- [ ] Mobile-friendly formatting
- [ ] No broken internal links
- [ ] Canonical URL set correctly
- [ ] Appropriate schema markup noted

### Final Review
- [ ] Read aloud for natural flow
- [ ] Check against competitor content
- [ ] Verify all claims and statistics
- [ ] Ensure brand consistency
- [ ] Get second pair of eyes review
