## SEO Techniques & Strategy

### 1. Technical SEO Foundation

#### Site Architecture
- **Flat hierarchy**: Max 3 clicks to any page
- **Clear URL structure**: Logical, keyword-rich paths
- **XML Sitemap**: Auto-generated, submitted to GSC
- **Robots.txt**: Properly configured crawl directives

#### Core Web Vitals
- **LCP**: < 2.5 seconds
- **FID/INP**: < 100ms
- **CLS**: < 0.1

#### Schema Markup
```json
{
  "Service Pages": "Service schema",
  "Location Pages": "LocalBusiness schema",
  "Blog Posts": "Article schema",
  "Case Studies": "Case Study schema",
  "FAQ Sections": "FAQPage schema",
  "Reviews": "Review schema"
}
```

### 2. On-Page SEO Strategy

#### Title Tag Formula
```
[Primary Keyword] | [Secondary Keyword] - GrowBytes
Example: "Digital Marketing Agency Dubai | SEO & PPC Services - GrowBytes"
```

#### Meta Description Formula
```
[Action verb] + [benefit] + [keyword] + [CTA]
Example: "Transform your online presence with Dubai's leading digital marketing agency. Expert SEO, PPC & social media services. Get a free consultation today!"
```

#### Header Structure
```
H1: Primary keyword (1 per page)
H2: Secondary keywords/topics
H3: Supporting points/features
H4: Sub-details if needed
```

#### Content Requirements
- **Service Pages**: 1,500-2,500 words
- **Location Pages**: 1,000-1,500 words
- **Blog Posts**: 1,500-3,000 words
- **Case Studies**: 800-1,500 words

PILLAR: /services/seo-organic-growth/
